# 최우영 주방 운영 시스템

주방 포지션별 업무 리스트와 체크리스트가 분리된 프리미엄 F&B 운영 앱입니다.

## 구조

- 홈: 주방 포지션 선택
- 포지션: 포지션별 전용 업무 리스트
- 체크리스트: 각 업무별 20개 체크 항목
- 100% 완료 시 저장 버튼 활성화

## 포지션

- 고기 다이
- 시코미 1
- 시코미 2
- 설거지
- 포장
- 마감

## 실행 방법

```bash
npm install
npm run dev
```

## GitHub / Vercel 업로드

1. ZIP 압축을 풉니다.
2. 폴더 안의 파일 전체를 GitHub 새 저장소에 업로드합니다.
3. Vercel에서 해당 GitHub 저장소를 Import 합니다.
4. Framework Preset은 Vite로 선택합니다.
5. Deploy를 누릅니다.
